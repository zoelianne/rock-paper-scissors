# rock-paper-scissors
Rock Paper Scissors console game for The Odin Project

## About
This is a console-based Rock Paper Scissors game where you play 5 rounds against the computer. The game keeps track of scores and announces a winner at the end.

## Features
- Random computer choice generation
- Case-insensitive user input
- Score tracking across 5 rounds
- Win/loss/tie detection

## What I Learned
- Functions and parameters
- If/else conditional logic
- For loops
- Using Math.random() for randomization
- String methods (.toLowerCase())
- Template literals
- Score increment operators (++)

## How to Play
1. Open `index.html` in a browser
2. Open the browser console
3. Enter your choice when prompted (rock, paper, or scissors)
4. Play 5 rounds and see who wins!
